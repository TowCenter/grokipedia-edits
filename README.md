# Grokipedia Edits Data

**Download CSV for data on suggested edits to Grokipedia here**:[https://grokipedia.s3.us-east-1.amazonaws.com/all_edits.csv](https://grokipedia.s3.us-east-1.amazonaws.com/all_edits.csv)

Read the story: [https://www.cjr.org/tow_center/grok-is-now-editing-itself-ai-x-twitter-elon-musk-xai-chatbot.php](https://www.cjr.org/tow_center/grok-is-now-editing-itself-ai-x-twitter-elon-musk-xai-chatbot.php)

The dataset includes a subset of suggested edits to Grokipedia, Elon Musk's Wikipedia alternative. Each row represents a suggested edit to a Grokipedia page, with Grok's approval/denial and reasoning. Data spans from the week of October 27, 2025, to January 12, 2026, and may be partially incomplete.
